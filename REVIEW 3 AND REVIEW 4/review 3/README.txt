How to Run
Setup Environment:

Install Apache Tomcat.
Ensure your IDE supports Servlets and JSP (e.g., Eclipse, IntelliJ).
Deploy the Application:

Package the project as a WAR file.
Deploy it to the webapps folder of Tomcat.
Access the Application:

Start Tomcat and navigate to:
Registration: http://localhost:8080/project-name/register.jsp
Profile: http://localhost:8080/project-name/profile.jsp
